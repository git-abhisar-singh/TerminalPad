Everything you launch, in one place.

Your agents and every CLI you've installed — searchable, beautiful, one keystroke away.

⌘

Spotlight search

Opens focused. Type to filter, ↑↓ to move, Enter launches the top match, Esc closes.

◳

Launchpad grid

A pure-dark panel — real Liquid Glass on macOS 26, a clean blur on older versions. A global hotkey (⌥⌘Space) summons it from anywhere.

⚡

Per-mode tiles

claude, claude --dangerously-skip-permissions, claude -c — each mode gets its own tile, icon, and color.

✦

Real brand logos

Sourced from Simple Icons, rendered to match light or dark. ~50 ship bundled; unknown tools fetch on demand and cache.

⟳

Instant auto-discovery

Scans Homebrew, npm, pipx, cargo, go & bun in parallel. Results cache to disk so the grid is instant on every launch.

{ }

JSON config

Add or edit agents in ~/.config/terminalpad/agents.json, hit reload. No rebuild, no ceremony.

Install in 30 seconds.

Requires macOS 14 (Sonoma) or later — Apple Silicon or Intel. Just the Swift toolchain — xcode-select --install is enough, no full Xcode.

# clone, build, and drop it in /Applications
$ git clone https://github.com/git-abhisar-singh/TerminalPad.git
$ cd TerminalPad
$ ./install.sh

You compile it yourself — so there's no Gatekeeper warning, and no $99 Apple Developer account required.

How it works.

1

Open

Hit ⌥⌘Space or click the menu-bar icon. The glass panel appears, search already focused.

2

Find

Type a few letters. Your agents and every installed CLI rank instantly by relevance and frequency.

3

Launch

Press Enter. TerminalPad opens a fresh Terminal window and runs the exact command — in the right folder, in your login shell.
